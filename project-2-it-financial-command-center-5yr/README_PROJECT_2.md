# Project 2: Nexora IT Financial Command Center

This folder contains a 5-year, Power BI-ready IT finance dataset and documentation package for building an executive IT Financial Command Center.

## Folder Contents
- `/data`: CSV files for fact and dimension tables
- `/docs`: dashboard requirements, DAX measures, data model, executive insights, and build guide
- `/assets`: architecture diagram and dashboard wireframes

## Dataset Span
2022 through 2026, monthly data across 60 months.

## Skills Demonstrated
IT FP&A, Power BI, Power Query, DAX, data modeling, executive reporting, vendor analysis, cloud cost analysis, headcount analytics, and forecast analysis.

## Recommended Next Step
Import the CSV files into Power BI Desktop and follow `/docs/power_bi_build_guide.md`.
