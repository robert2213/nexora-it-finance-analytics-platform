# Nexora IT Financial Command Center — Project Summary

## Business Problem
IT leaders need a single command center to understand budget, actuals, forecast, headcount, external labor, vendor spend, cloud growth, and project investment across a complex IT organization.

## Solution
This project provides a 5-year Power BI-ready dataset and supporting documentation for an enterprise IT finance dashboard. The model is designed around a scalable star schema with fact tables for actuals, budget, forecast, headcount, vendors, cloud, external labor, and projects.

## Tools Used
- Power BI
- Power Query
- DAX
- CSV source files
- Star schema modeling
- AI-assisted project documentation and dataset generation

## Skills Demonstrated
Financial analysis, IT FP&A, data modeling, business intelligence, executive reporting, variance analysis, forecast analysis, vendor management, and dashboard requirements design.

## Business Value
The command center reduces manual reporting friction and gives leaders a clean view of what changed, why it changed, and where action is needed.
