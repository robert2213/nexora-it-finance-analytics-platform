# Dashboard Requirements

## Page 1 — CIO Executive Summary
KPIs: Total Budget, Total Actuals, Total Forecast, Year-End Projection, Variance $, Variance %, Spend by Business Unit, Spend by Category, Top 5 Variance Drivers, Executive Commentary.

## Page 2 — Budget vs Actuals
Monthly actuals vs budget trend, YTD actuals, YTD budget, YTD variance, variance by cost center, variance by GL category, drill-through to transaction detail.

## Page 3 — Forecast & Projection
3+9, 6+6, and 9+3 forecast comparison, current forecast vs budget, full-year projection, forecast accuracy, remaining forecast, risk/opportunity flags.

## Page 4 — External Labor
External labor spend, contractor count, average hourly rate, vendor spend, resource ramp-up/down, labor by cost center, labor by project, contract end date risk.

## Page 5 — Vendor Spend
Top vendors by spend, vendor YoY growth, vendor budget variance, vendor by category, concentration risk, renewal or contract flag.

## Page 6 — Cloud Spend
Cloud spend by provider, Azure vs AWS trend, cloud spend by cost center, cloud spend by project, forecast vs actuals, optimization opportunity.

## Page 7 — Headcount
Approved positions, filled positions, open positions, vacancy rate, labor cost trend, headcount by business unit, headcount by cost center.

## Page 8 — Project Spend
Project budget, project actuals, project forecast, capital vs operating split, project variance, project status, spend by project owner.

## Page 9 — Drill-through Detail
Transaction date, month, business unit, cost center, GL account, vendor, project, scenario, amount, notes.
